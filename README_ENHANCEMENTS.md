## ❓ FAQ

### Q: Do I need to change anything to use this?
**A**: No! Your existing files work as-is. Just open and enjoy the enhanced visuals.

### Q: Are there breaking changes?
**A**: Absolutely not. 100% backward compatible. All existing features work unchanged.

### Q: How do I display the percentages?
**A**: They display automatically. Make sure your tasks have a `progress` value (0-100).

### Q: Can I export as image?
**A**: Not yet, but you can take a screenshot or this feature is planned.

### Q: Is this production-ready?
**A**: Yes! Built with Vite, tested thoroughly, and ready to deploy.

### Q: How do I create the orthogonal connectors?
**A**: They're automatic! The `calculatePath` function in GanttChart.tsx creates them.

### Q: Will my custom projects work?
**A**: Yes! Percentages default to 0% if not specified. Update them anytime.
